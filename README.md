# Patient management system - frontend 

## Description 
   1.this is the frontend of the patient management system built using HTML , CSS , JavaScript 
   2.it provides a user interface for interacting with the backend 

## Features 
   1.user authenticztion and authorization(login and register pages )
   2.add, update, delete patients
   3.manages patients medical records 
   4.error handling 
   5.JWT token handling(stored in local storage) 

## Technologies used 
   1.HTML
   2.CSS
   3.Vanilla js
   

## Pages 
   1.register.html (new user registration) 
   2.login.html (user login) 
   3.patient.html (main dashboard) 

## Backend API 
   1.this frontend communicates with a spring bot backend. link : https://github.com/SARAH-BENABED/first-API-demo-spring-bot.git

## UI Features 
   1.centered container using flexbox
   2.backgrounf image
   3.hover effect on buttons 
   4.displaying patietns info as a grid 
   5.clean and simple design 

## How to run the project
   1.open(register.html) in your browser 

## Notes 
  1.make sure backend is running (API requests will fail otherwise) 
  2.CORS must be enabled on backend
